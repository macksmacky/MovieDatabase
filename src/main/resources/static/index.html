Welcome to the MacMillan Learning Movie Database

The Following Endpoints are available


Request Type Request Path Information
Get Request /api/v1/timeOfDay Returns the Current Server time in 12 hour format.
Post Request /api/v1/movie Accepts Movie JSON. Adds New Movie to the Database. Validates Rating 0-100 and Unique Names.
Put Request /api/v1/movie Accepts Movie JSON. Updates existing Movie By Id. Error If Movie is not found.
Delete Request /api/v1/movie/{id} Deletes movie by id. Error if movie not found by Id.
Delete Request /api/v1/movie/{name} Deletes movie by name. Error if movie not found by Name.
Get Request /api/v1/movie/list Returns a list of all saved Movies


H2 console Reachable at:

localhost:8080/h2-console/

JSON Object Example for Put and Post Requests

{ "id": 1, "name": "Blade Runner", "genre": "Science Fiction", "rating": 10 }
